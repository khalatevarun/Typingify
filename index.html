Rules and tips

The timer starts as soon as you start typing in the text area.

To stop the current recording, press 'Enter' and the score will recored in the table.

To restart the recording, press 'Escape' and the parameters will be set back to inital values.

You can't copy the text displayed nor can you move the cursor using arrow keys

Words = Total Characters Typed / 5

Errors = No. of Incorrect Characters Typed

Accuracy = ( Correct Characters Typed / Total Characters Typed ) * 100 %

CPM = ( Correct Characters Typed / Duration ) * 60

WPM = CPM/5
