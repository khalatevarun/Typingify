About Typingify

Typingify is a website to test your typing skills.

This is how the paramteres are calculated:

Words = Total Characters Typed / 5

Errors = No. of Incorrect Characters Typed

Accuracy = ( Correct Characters Typed / Total Characters Typed ) * 100 %

CPM = ( Correct Characters Typed / Duration ) * 60

WPM = CPM/5
